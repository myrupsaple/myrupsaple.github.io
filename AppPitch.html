Motor Trend MPG Predictor

Quasi-Poisson logarithmic models for predicting MPG are generated using the following as predictors:

  • Horsepower
  • Weight
  • Transmission Type

What does this application offer?

  • The user can select a preset combination of variables to manipulate
  • The model will be plotted alongside the mtcars dataset
  • The predicted MPG of the user's model car will be read out

Example

A model using 150 HP and manual transmission as predictors.

[1] "The predicted MPG for your model car is:  22.08  MPG"

Another Example

Why Use This Modeling Device?

People care about MPG ratings

  • Gas prices are constantly rising
  • Understanding how certain variables affect MPG may help people when looking for a new car
  • This information may be useful for manufacturers as they design new cars
